Welcome to the Java documentation of the Keypop Generic Card Jvm API, part of the Keypop open-source project hosted by the Eclipse Foundation.

Aligned with the Terminal Generic Card UML specifications proposed by the Calypso Networks Association (available here), it defines the interfaces needed to select and operate transactions with ISO/IEC 7816 cards.

Note: as you navigate, you might observe that certain interfaces either reference or extend elements from the Keypop Reader Java API.
